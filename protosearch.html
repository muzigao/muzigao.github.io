PROTOSEARCH
Columbia University, New York, NY

Description

Facebook has created an extraordinarily large social graph of the web, intricately connecting people and a variety of things. Currently, users may get a linear list of results by using the search box. However, we wonder whether there is a better way to present this information.

In this project called PROTOSEARCH, we explore a visualized way to present search results where the emphasis is on connections and social relationships between users and those results.

Cooperated with Chatura Atapattu, Aditya Pipersenia and Xiaolong Jiang.

Responsibility

• Analyzed features and conducted user study, establishing persona, scenario and paper prototype.

• Implemented UI elements and components of search result with Adobe Flash Builder.

• Conducted usability test and heuristic evaluation.
